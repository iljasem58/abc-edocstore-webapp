License information
Application ID contact_support

The Current eDocStore Application ID. NB! The Application ID depends on the accounts of the CosmosDB and the Azure Blob Storage. If any of them changes (e.g. by changing connection string), then the Application ID WILL change and the license bought for the previous Application ID will not work (it will have 'Invalid' status).

License status contact_support
Indicates current license status.
  • License is missing! - the license code element ('licenseKey') or it's value is not found in the Application configuration.
  • Active - the license is valid and currently effective.
  • Sheduled - the license is valid, but it's effective period is not started yet.
  • Expired - the license is valid, but it's effective period is ended. To be able to continue the usage the service, a new license must be provided.
  • Invalid - the license is invalid, detailed exception can be found in application logs.
